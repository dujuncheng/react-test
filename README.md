# 您好，面试官

## 如何启动

1. clone 项目到本地
2. npm install 
3. npm run start


## Step 1: 安装nodejs   ` 完成 `
能启动本地服务器。
您可以在`/test`目录下，执行 `node app.js`, 注意：本地的8080端口不要被占用

## Step 2: 安装react   ` 完成 `
您可以在`/test`目录下，执行 `npm run start `, 访问localhost:8081，即可看到hello,world

## Step 3: 使用案例, 把日期选择范围区间设为 当日 至 2017-08-31。  ` 完成 `
1.  `cd ./antd-demo`
2. `npm run start`
3. `访问localhost:8000`
